# Product Specifications

## Discount Codes
- Valid discount code: SAVE15
- SAVE15 applies 15% discount to total order
- Invalid codes should show error message
- Discount is applied before shipping costs

## Shipping Methods
- Standard Shipping: Free
- Express Shipping: $10.00
- Shipping costs are added after discount application

## Form Validation
- All fields marked with * are required
- Email must be in valid format (user@domain.com)
- Validation errors shown in red text
- Form must be valid to proceed with payment

## Payment Flow
- User must complete all required fields
- Payment successful message shown upon valid submission
- System supports Credit Card and PayPal
